## Contribution guide
Feel free to contribute in this project.

## Commands

-   **Build** `npm run build`
-   **Extract plugin (xynity-blocks.zip)** `npm run extract`

## Frontend Library/Framework

-   React
-   Webpack
-   Babel
-   Tailwindcss
-   Postcss
-   etc...

## File structure

-   Frontend development dir - `/src`  
    build dir - `/dashboard`
