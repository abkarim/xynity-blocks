# xynity-blocks

Extends Block Themes functionality and Adds Wordpress blocks.

## Recommendations

-   **Theme** Twenty Twenty Three

## Commands

-   **Build** `npm run build`
-   **Extract plugin (without dev dependencies)** `npm run extract`

## Frontend Library/Framework

-   React
-   Webpack
-   Babel
-   Tailwindcss
-   Postcss

## File structure

-   Frontend  
    development dir - `/src`  
    build dir - `/dashboard`
